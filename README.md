Task 3: Exploratory Data Analysis (EDA)

Project Overview-
This repository contains the Exploratory Data Analysis for two different datasets: **Iris** (Numerical) and **Netflix** (Categorical). The goal was to visualize data patterns, identify outliers, and understand feature relationships.

Repository Structure -
Iris_EDA_Project/: Focuses on biological measurements and correlation heatmaps.
Netflix_EDA_Project/: Focuses on content trends, release years, and categorical distributions.

Tools Used -
Pandas: Data manipulation.
Matplotlib & Seaborn: Advanced data visualization.

Key Deliverables-
1. EDA Notebooks: Containing histograms, box plots, and heatmaps.
2. Visual Insights Report: A detailed breakdown of findings for each chart.
